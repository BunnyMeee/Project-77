Visual Studio

Visual Studio is a program to code. Here is how to set it up: 1st, you must download visual studio, and click on the agreements that pop up. Next, On the left pane, the last icon, click it, and search "Live Server" and choose the 1st option that comes. Create a folder, and open the folder in visual studio. Now beside the left pane, it shows your folder name. Under it, tap with 2 fingers, and a text box will appear. Now, type the language you will be using! Now, You are done setting up Visual Studio.

Java Script Console

The Console screen is very easy to find. Tap With 2 fingers, and Click "inspect", Or do Ctrl+Shift+I. Something on the right will pop up, and there is console on the top. Click that, that is how to get to the console screen. You can also put stuff on the console screen, using java script. First, Create a variable. Now, type "console.log()", and place your variable name in the parenthesis. Now, you can check the console screen, and what you put in the console log, shows there. The console log is used to place how to do things, and to put stuff.
